# Security Policy

## Supported Versions

We are committed to ensuring the security of our API. Below are the versions of the project that are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

We take all security bugs in this project seriously. If you discover a security vulnerability, please report it to us as soon as possible. We will work with you to make sure that we understand the scope of the issue and that we fully address your concern.

To report a security vulnerability, please send an email to [your-email@example.com](mailto:your-email@example.com) with a detailed description of the vulnerability and steps to reproduce it. **Please do not create a public GitHub issue.**

We will respond to your report within 48 hours and will do our best to patch the vulnerability as soon as possible. We will also credit you for the discovery unless you prefer to remain anonymous.

Thank you for helping keep this project safe.
